# meteor starter kit V0.1.5

A simple kit to start experimenting with Apollo, Meteor and React.

### Includes: #
- Apollo server
- Apollo client
- Express
- React
- Accounts UI, Basic & password
* check `package.json` for specific versions


### Running it

```
meteor npm install
meteor
```
